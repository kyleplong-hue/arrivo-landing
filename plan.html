Demo App (Molino Nobile)

The Tuscany demo is the hero. It needs to feel like a real product, not a prototype.

Performance: React via CDN + Babel is slow

Currently loading React, ReactDOM, and Babel from CDN and compiling JSX in the browser. Page load is sluggish. Options: (A) Pre-compile to vanilla JS, (B) Switch to Preact + HTM (no build step, much smaller), (C) Keep as-is for demo purposes.

Performance
High Priority

AI Chat needs an API key proxy

The chat tab currently calls the Anthropic API directly from the browser. This means either: (A) it doesn't work without a key, or (B) someone's key is exposed in the source. Need a simple proxy/edge function.

Technical UX
High Priority

Guest link sharing is fragile

Guest links encode the entire itinerary in base64 URL params. Works but: URLs are enormous, can break in email clients, and there's no server-side persistence. Fine for demo, bad for production.
